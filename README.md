# AI-Multiverse-Upgrade
AI Multiverse is a Streamlit-based chatbot application powered by the Google Gemini API. Users can chat with different AI personalities, customize response style and length, view chat statistics, and download conversation history.

## ✨ Features

* 🤖 Multiple AI Personalities (Iron Man, Batman, Sherlock Holmes, Harry Potter, Elon Musk, Virat Kohli, Deadpool, and more)
* 🎭 Custom Response Styles (Friendly, Funny, Professional, Motivational, Short)
* 📏 Adjustable Response Length (Short, Medium, Long)
* 🎲 "Surprise Me" Prompt Generator
* 💬 Chat History
* 📥 Download Conversation as a Text File
* 📊 Chat Statistics (Messages, Characters, Words)
* ⚡ Fast Responses with Google Gemini API
* 🎨 Modern Streamlit User Interface

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Gemini API
* python-dotenv
* Random
* Datetime

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/bhoomika-cj/AI-Multiverse-Upgrade1.git
```

2. Navigate to the project folder

```bash
cd AI-Multiverse-Upgrade1
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Create a `.env` file and add your Gemini API key

```env
GEMINI_API_KEY=YOUR_API_KEY
```

5. Run the application

```bash
streamlit run app.py
```
