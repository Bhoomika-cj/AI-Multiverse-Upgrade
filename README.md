# 🌌 AI Multiverse Upgrade

AI Multiverse Upgrade is an interactive chatbot application built using **Python**, **Streamlit**, and the **Google Gemini API**. It allows users to chat with multiple AI personalities, customize response styles and lengths, view chat statistics, and download conversation history through a clean and modern interface.

## 🚀 Features

* 🤖 Multiple AI Personalities
* 🎭 Custom Response Styles
* 📏 Adjustable Response Length
* 🎲 Surprise Me Prompt Generator
* 💬 Chat History
* 📥 Download Conversation
* 📊 Chat Statistics
* ⚡ Powered by Google Gemini API
* 🎨 Modern Streamlit User Interface

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* python-dotenv

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/Bhoomika-cj/AI-Multiverse-Upgrade.git
```

2. Go to the project folder

```bash
cd AI-Multiverse-Upgrade
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Create a `.env` file and add your Gemini API key

```env
GEMINI_API_KEY=YOUR_API_KEY
```

5. Run the application

```bash
streamlit run app.py
```
